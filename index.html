This is a complete restaurant ordering solution.

This project includes :

Mobile tablets App with Sencha Touch 2

  • a mobile app for android and iPad tablets showing a restaurant menu on a nice interface, letting customers see products, add them to a order list and send their order to the restaurants kitchens. This app is entirely written with Sencha Touch 2 JavaScript/HTML5 framework.

Backbone.js boilerplate require.js Twitterbootstrap administration app

  • a complete menu administration app letting restaurants owners manage the content, products, menus, sets, waiters, see stats informations about which products has been seen most and more to come. this app is written with Backbone.js boilerplate, require.js, themed with Twitter Bootstrap, and includes socket.io for real time notifications about waiters call from customers or orders made.

node.js/mongodb/socket.io for data storage, data delivering and real time notifications

  • a node.js/mongodb/socket.io serveur that runs and delivers the data to the differents apps.
